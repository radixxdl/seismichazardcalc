Seismic Hazard Calculator (alpha)

The Seismic Hazard Calculator is powered by OpenSHA. The source code for OpenSHA can be accessed at the OpenSHA website.

This application has been made accessible to the public in order to demonstrate the benefits of site-specific hazard analyses.
These benefits have been discussed in the Bradley (2015) paper, as published in volume 48, issue 2 of the NZSEE bulletin.

This is an alpha release, meaning that the application is still in development, and not all of the core features are present.
There is no version number currently assigned to this release, but the source code can be accessed via GitHub.
For now, this tool has been made available on the web for testing purposes only.
Disclaimer: While the Seismic Hazard Calculator is available for public use, it must be emphasised that no liability for its use is accepted.

Variable Inputs

Street address to look up on Google Maps


The address could not be found on Google Maps. 
Latitude in degrees North

The valid range of latitudes for NZ is -48 to -34. 
Longitude in degrees East

The valid range of longitudes for NZ is 166 to 179. 
Average shear-wave velocity to 30 m
This is the time-averaged shear-wave velocity between 0 and 30 metres depth.
For more information visit the OpenSHA Glossary.

The valid range of vs30 is 150 to 3000. 
Depth where shear-wave velocity is 1.0 km/sec
This is the depth (km) to where shear-wave velocity is equal to 1.0 km/sec.
For more information visit the OpenSHA Glossary.

This value can be estimated through an empirical relationship with Vs30. (Chiou and Youngs, 2008)
To use this estimated value, check "use default".


The valid range of z1.0 is 0 to 1000. 
Period with which spectral acceleration is to be determined

 

Fixed Inputs

Stirling et al. (2012)
 
Bradley (2013)
 
50 years
 

Output Options

Disaggregation


The valid range of disaggregation value is 0.001 to 0.95.
Note: For each option selected, the disaggregation function will take up to two minutes to process.
